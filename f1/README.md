📱 Anonymous Feedback App (Flutter)

A cross-platform mobile and web application built using Flutter that allows users to submit feedback anonymously. Designed for colleges, hostels, and organizations to encourage honest communication without revealing identity.


🚀 Features

🕵️ Anonymous feedback submission
📂 Categorized feedback (Complaints, Suggestions, Issues)
📱 Cross-platform support (Android, iOS, Web, Desktop)
⚡ Fast and responsive UI with Flutter
🔒 Privacy-focused (no personal data stored)
🧑‍💼 Admin view (if implemented)


🛠️ Tech Stack

Framework:Flutter
Language:Dart
Database:Firebase
